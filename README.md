# mumsmums

## Running locally

Build all sources:

    ./scripts/mumsmums build

Run all tests:

    ./scripts/mumsmums test

Start the app:

    ./scripts/mumsmums start

Stop the app:

    ./scripts/mumsmums stop

## Running in Docker

Start the app:

    ./scripts/mumsmums start --docker

Stop the app:

    ./scripts/mumsmums stop --docker
